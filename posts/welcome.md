# 블로그에 오신 것을 환영합니다!

이 블로그는 **Next.js**와 **마크다운**을 사용하여 만들어졌습니다. 개발과 취미에 관한 다양한 주제를 다룰 예정입니다.

## 주요 기능

- 마크다운 기반 포스트 작성
- 코드 하이라이팅 지원
- 다크 모드 지원
- 반응형 디자인

## 사용 방법

`posts` 디렉토리에 마크다운 파일을 추가하면 자동으로 블로그 포스트로 등록됩니다.

### 예시 코드

```typescript
// TypeScript 예시
function greet(name: string): string {
  return `안녕하세요, ${name}님!`;
}

console.log(greet('방문자'));
```

```javascript
// JavaScript 예시
const posts = getAllPosts();
posts.forEach(post => {
  console.log(post.metadata.title);
});
```
